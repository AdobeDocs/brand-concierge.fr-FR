# En savoir plus sur les identifiants de flux de données

Découvrez comment l’identifiant du flux de données connecte les événements Web SDK de votre site à Adobe Brand Concierge dans Adobe Experience Platform. Vous découvrirez pourquoi l’identifiant est obligatoire, comment il achemine les données vers le sandbox et les jeux de données appropriés et comment les équipes utilisent des flux de données distincts pour le développement, l’évaluation et la production.
